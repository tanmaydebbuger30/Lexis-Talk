# Lexis-Talk
LexiTalks is an end to end Retrieval Augmented Generation (RAG) system that supports both voice and text interaction. The application processes PDF documents, builds a Legal-BERT powered vector database using ChromaDB, and answers user queries using GPT-4o with contextual grounding.

 Voice input via Whisper (ASR)

 Text-based Q&A

 English & Hindi support

 Legal-BERT embeddings

 PDF document retrieval with ChromaDB

 OpenAI TTS audio responses

 Fuzzy + keyword boosted retrieval scoring

This project demonstrates full stack LLM system design including embedding pipelines, vector storage, contextual prompting, multilingual translation, and interactive UI deployment using Gradio.
